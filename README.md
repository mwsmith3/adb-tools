<!-- Plugin description -->
Provides a tool-window for executing common adb-commands on connected Android devices.
Commands:

- Start App
- Restart App
- Stop App
- Uninstall App
- Clear Data and Restart App
- Enable Developer Options
- Toggle Talkback
- Set Device Screen Size And Density
- Set Animation Scales
<!-- Plugin description end -->

## Installation
 
- Manually:

  Download the [latest release](https://github.com/mwsmith3/adb-tools/releases/latest) and install it manually using
  <kbd>Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>
